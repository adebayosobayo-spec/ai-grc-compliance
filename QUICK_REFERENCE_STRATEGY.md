## WEEKLY DECISION FRAMEWORK

### If You Have 15 Minutes:
```
1. Revenue check: MRR target on pace? (Aiming +30% MoM in Year 1)
   → If behind: Activate partnership channel? Increase ad spend?

2. Customer happiness: Any churn? Any 5-star reviews?
   → If high churn: Product issue or market fit issue?

3. Team velocity: Stories shipped this week?
   → If slower: Blocker? Scope creep? Hire?

4. Top 3 priorities for next week?
   → Alignment. Go.
```

### If You Have 1 Hour:
```
METRIC CHECK (20 min)
├─ MRR, paid customers, churn, NPS
├─ Website traffic, organic leads, partnership pipeline
└─ Question: On track or need adjustment?

CUSTOMER INTERVIEWS (30 min)
├─ Call 1-2 recent customers: How's it going? What's working? Missing?
├─ Call 1 churned customer: Why'd you leave?
└─ Insight: Product, pricing, or market fit issue?

PRIORITIES (10 min)
├─ What's blocking revenue growth?
├─ What's our #1 next move?
└─ Team alignment on that priority.
```

## DECISION TREE: When Things Get Confusing

```
Q: "Should we pivot?"
├─ IF <$10K ARR after 90 days:
│  ├─ Check: Is it a product issue? (Talk to customers)
│  ├─ Check: Is it a market issue? (Wrong target segment?)
│  ├─ Check: Is it a go-to-market issue? (Not reaching buyers?)
│  └─ Action: Double down on ONE of the three above
│
└─ IF $20K+ ARR and growing:
   └─ Don't pivot. Just accelerate.

Q: "Should we raise capital?"
├─ IF you can reach $250K ARR by Dec 2026 on current runway:
│  └─ Don't raise. Stay independent, keep equity.
│
└─ IF you need to hire faster than revenue covers:
   ├─ Raise $500K-1M Series A
   └─ Hire aggressively, expand to enterprise sales

Q: "Should we do feature X?"
├─ IF customer is paying $100+/mo and requesting it:
│  └─ Do it. (Paying customer requests = signal)
│
├─ IF free user requesting it:
│  └─ Not yet. (Free users don't drive revenue)
│
└─ IF it aligns with multi-framework strategy:
   └─ Consider it in Phase 2-3

Q: "Is product-market fit real?"
├─ IF retention > 90% (churn <5%) AND NPS > 50:
│  └─ YES. Scale. (You have PMF)
│
├─ IF retention 70-90% OR NPS 40-50:
│  └─ MAYBE. Fix one thing, measure again.
│
└─ IF retention < 70% AND NPS < 40:
   └─ NO. Find the blocker.
```

## RED FLAGS (Course Correction Needed)

```
🚨 Revenue Stalling
   Problem: <$5K MRR by week 4
   Action: Customer interviews (what's blocking buyers?)
   Options: Lower pricing, shift to partnerships, vertical focus

🚨 Churn Spiking
   Problem: >5% monthly churn
   Action: Exit interviews (why are they leaving?)
   Options: Improve onboarding, add feature, better support

🚨 Team Burnout
   Problem: Low velocity, frustrated faces
   Action: Retrospective (too many priorities? wrong skills?)
   Options: Hire help, narrow scope, extend runway

🚨 Partners Not Converting
   Problem: Signed LOI but no leads by week 6
   Action: Ask partner (are they promoting? Wrong fit?)
   Options: Activate new partnership, increase organic

🚨 CAC Too High
   Problem: Spending $3K+ to acquire $500/mo customer
   Action: Analyze channel (organic vs paid vs partnership)
   Options: Kill expensive channels, double down on cheap ones
```
